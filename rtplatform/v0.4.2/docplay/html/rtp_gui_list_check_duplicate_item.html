int rtp_gui_list_check_duplicate_item

( void* vplist,
  char* item)

Check if an item is already contained in an item contained in a list or menu object.

Documentation

Checks if the string exactly matches a string that was passed in the item argument to one of the forms of rtp_gui_list_add_item(). The item argument is the text that is displayed when the list is displayed. This function may be used to identify and discard duplicate information like the device identifier information in rebroadcasts of SSDP advertisements.

Parameters:
vplist - Handle returned from rtp_gui_list_init()
item - String to check for a match
Returns:
1 if the an exact match for item is already in the list. 0 otherwise.
Preconditions:
rtp_gui_list_init must be called first.
