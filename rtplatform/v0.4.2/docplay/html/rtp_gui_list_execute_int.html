int rtp_gui_list_execute_int

(void* plist)

Execute a list and return the item id of the selected item.

Documentation

Displays the items in the list (see rtp_gui_list_execute_element()) and returns the itemid assigned by rtp_gui_list_add_int_item(). Call rtp_gui_list_execute_void() instead to process the menu if void * values were assigned by rtp_gui_list_add_item().

Parameters:
plist - Handle returned from rtp_gui_list_init()
Returns:
An item id value passed in rtp_gui_list_add_int_item() or zero if none selected. (all greyed).
Preconditions:
rtp_gui_list_init must be called first, rtp_gui_list_add_item() or rtp_gui_list_add_int_item() must have been called one or more times
