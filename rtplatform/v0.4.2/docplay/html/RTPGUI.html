RTPGUI

User Interface Services using generic interface. If a GUI is available
the generic implementation may be replaced by an implementation that utilizes
the native UI instead of terminal IO.
rtp_gui_list_init
Create a list or menu object.
rtp_gui_list_destroy
Release all memory used by a list or menu object.
rtp_gui_list_add_item
Add an text entry and a void * value a list or menu object.
rtp_gui_list_add_int_item
Add an text entry and an integer value to a list or menu object.
rtp_gui_list_size
Retrieve the count of entries in a list or menu object.
rtp_gui_list_check_duplicate_item
Check if an item is already contained in an item contained in a list or menu object.
rtp_gui_list_execute_item
Display the content contained in a list or menu object. And retrieve user input.
rtp_gui_list_execute_void
Execute a list and return the item value of the selected item.
rtp_gui_list_execute_int
Execute a list and return the item id of the selected item.
rtp_gui_conout
Display a status line.
rtp_gui_conout_two
Display two status items on the same line.
rtp_gui_prompt_text
Prompt the user for input.
These functions sometimes need to be ported.

Depends on: RTPTERM
