RTPWCS

Wide String Library Services
rtp_wcsicmp
Compare characters of one unicode buffer with another.
rtp_wcsistr
Find a unicode string within a string.
rtp_wcsncmp
Compare characters of one unicode buffer with another.
rtp_wcsnicmp
Compare characters of one unicode buffer with another.
rtp_wcscat
Copies the source string to the end of the destination string.
rtp_wcschr
Find a character within a string.
rtp_wcscmp
Compare characters of one string with another.
rtp_wcscpy
Copies the source string to the destination.
rtp_wcslen
Finds the length of a string.
rtp_wcsncpy
Copies the source string to the destination.
rtp_wcsrchr
Find the last occurance of a character within a string.
rtp_wcsstr
Find a string within a string.
rtp_wcstol
Find a string within a string.
rtp_wcsncat
concatonate at most n char from src to end of targ; targ is null terminated
rtp_wcsspn
return length of prefix of str consisting of characters in chrs
rtp_wcscspn
return length of prefix of str consisting of characters NOT in chrs
These functions DO NOT need to be ported.
