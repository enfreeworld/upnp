void* rtp_gui_list_add_item

(void* vplist, char* item,
  void* itemvalue, int indentlevel,
  int isgreyed, int style )

Add an text entry and a void * value a list or menu object.

Documentation

Adds text and to a list or object. Also provides a void * value to attach to the item that will be returned when the item is selected.
If rtp_gui_list_execute() is called itemvalue will be returned.
If rtp_gui_list_execute_handle() is called the handle to the item is returned.

Parameters:
vplist - Handle returned from rtp_gui_list_init().
item - Text to display for the choice.
itemvalue - void *value to return when the item is selected.
indentlevel - Spaces to indent when text is displayed.
isgreyed - 1 if the item may not be selected.
style - Additional style attributes.
RTP_LIST_ITEM_STYLE_NOBREAK - Display this and the next list item side by side.
RTP_LIST_ITEM_STYLE_SUBMENU - Itemvalue is a menu or list handle returned from rtp_gui_list_init().
RTP_LIST_ITEM_STYLE_COLLAPSED - Don't display the submenu/sublist.
RTP_LIST_ITEM_STYLE_EXPANDED - Expand and display the submenu/sublist.

Returns:
A handle to the menu or list entry object.
Preconditions:
rtp_gui_list_init must be called first.
