int UPnP_ControlGetServicesByType

( UPnPControlDeviceHandle deviceHandle,
  UPnPControlServiceIterator* i,
  UPNP_CHAR* serviceType )

Enumerate the services of a certain type on a device

Documentation

Initializes a UPnPControlServiceIterator to enumerate all the services of the given type (as defined by the UPnP Forum) on the given device.

Generates: (no events)

Parameters:
deviceHandle - handle returned by \Ref {UPnP_ControlOpenDevice}
i - uninitialized \Ref {UPnPControlServiceIterator} to use for this enumeration
serviceType - type of service to enumerate
Returns:
error code
See Also:
UPnP_ControlGetServices, UPnP_ControlNextService, UPnP_ControlServiceIteratorDone
