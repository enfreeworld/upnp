int UPnP_ControlUnsubscribe

( UPnPControlDeviceHandle deviceHandle,
  UPNP_CHAR* serviceId,
  void* userData,
  UPNP_BOOL waitForCompletion )

Cancel a subscribed service

Documentation

If waitForCompletion is UPNP_TRUE, this function waits for the unsubscribe operation to complete before returning. Otherwise, it returns immediately and a UPNP_CONTROL_EVENT_SUBSCRIPTION_CANCELLED event is generated when the operation completes.

Generates: UPNP_CONTROL_EVENT_SUBSCRIPTION_CANCELLED

Parameters:
deviceHandle - handle returned by \Ref {UPnP_ControlOpenDevice}
serviceId - id of service to unsubscribe from
userData - passed to callback as userRequestData for generated events
waitForCompletion - see above description
Returns:
error code
