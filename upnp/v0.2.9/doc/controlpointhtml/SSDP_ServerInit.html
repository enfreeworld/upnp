SSDP_INT32 SSDP_ServerInit

( SSDPServerContext* ctx,
  SSDP_UINT8* ipAddr,
  SSDP_INT16 ipType,
  const SSDP_CHAR* serverId,
  SSDPCallback cb, void* cookie)

SSDP server initialization routine.

Documentation

This routine starts up SSDP services by creating a UDP socket, getting the ssdp multicast membership for the socket and initlializing ssdp context.

Parameters:
ctx - pointer to an uninitialized SSDP context structure
ipAddr - IP address of the host to bind the UDP socket to, if a NULL is supplied, the UDP socket is bound to the local IP address
ipType - ip version 4 or ipversion 6
serverId - String holding platform name
cb - SSDP Callback routine
cookie - cookie(runtime) to be stored in ssdp context to be passed later to ssdp callback
Returns:
error code
