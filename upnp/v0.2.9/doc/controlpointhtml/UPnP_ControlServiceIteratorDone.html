void UPnP_ControlServiceIteratorDone

( UPnPControlServiceIterator* i )

Clean up when done enumerating services.

Documentation

Must be called when done enumerating services using UPnP_ControlNextService.

Generates: (no events)

Parameters:
i - UPnPControlServiceIterator initialized by \Ref {UPnP_ControlGetServices} or \Ref {UPnP_ControlGetServicesByType}
Returns:
nothing
See Also:
UPnP_ControlGetServices, UPnP_ControlGetServicesByType, UPnP_ControlNextService
