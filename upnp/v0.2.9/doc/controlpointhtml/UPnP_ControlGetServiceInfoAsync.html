int UPnP_ControlGetServiceInfoAsync

( UPnPControlDeviceHandle deviceHandle,
  UPNP_CHAR* serviceId,
  void* userData )

Asynchronous get detailed information about a service.

Documentation

This function retrieves the service description document (SCPD) from the given device and parses it into an XML DOM structure. The IXML_Document is passed back to the control point through a UPNP_CONTROL_EVENT_SERVICE_INFO_READ event.

This function will return immediately; an event from the list below is sent to the control point when the SCPD has been downloaded, or the operation fails due to some error.

Generates: UPNP_CONTROL_EVENT_SERVICE_INFO_READ, UPNP_CONTROL_EVENT_SERVICE_GET_INFO_FAILED

Parameters:
deviceHandle - handle returned by \Ref {UPnP_ControlOpenDevice}
serviceId - id of service to get detailed info for
userData - passed to callback as userRequestData for generated events
Returns:
error code
