void UPnP_ControlSetServiceSubscriptionExpireWarning

( UPnPControlDeviceHandle deviceHandle,
  UPNP_CHAR* serviceId,
  UPNP_INT32 warningMsec )

Sets the time offset before subscription expiration at which a warning event will be generated.

Documentation

This function should be called, if desired, before subscribing to the given service.

Parameters:
deviceHandle - handle returned by \Ref {UPnP_ControlOpenDevice}
serviceId - id of service
warningMsec - time offset to generate warning, in milliseconds
Returns:
nothing
See Also:
UPnP_ControlSubscribe, UPnP_ControlUnsubscribe
