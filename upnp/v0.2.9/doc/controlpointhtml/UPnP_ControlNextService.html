UPNP_CHAR* UPnP_ControlNextService

( UPnPControlServiceIterator* i )

Enumerate the next service on the device.

Documentation

Returns the unique serviceId of the next service instance in the given enumeration (initialized by UPnP_ControlGetServices or UPnP_ControlGetServicesByType). The string returned by this function must not be modified in any way, and is valid until UPnP_ControlDeviceClose is called for this device.

Generates: (no events)

Parameters:
i - UPnPControlServiceIterator initialized by \Ref {UPnP_ControlGetServices} or \Ref {UPnP_ControlGetServicesByType}
Returns:
serviceId if there is a next service; otherwise NULL
See Also:
UPnP_ControlGetServices, UPnP_ControlGetServicesByType, UPnP_ControlServiceIteratorDone
