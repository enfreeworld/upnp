int UPnP_ControlGetDeviceInfo

( UPnPControlDeviceHandle device,
  UPnPControlDeviceInfo* info )

Get information for an open device.

Documentation

Populates the fields of the specified UPnPControlDeviceInfo structure with various information about the given device, such as device type, UDN, manufacturer, model number, etc.

Generates: (no events)

Parameters:
device - handle returned by UPnP_ControlOpenDevice
info - UPnPControlDeviceInfo to populate
Returns:
error code
See Also:
UPnP_ControlGetServiceOwnerDeviceInfo
