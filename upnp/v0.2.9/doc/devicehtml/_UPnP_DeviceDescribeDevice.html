void _UPnP_DeviceDescribeDevice

( UPnPDevice* device,
  IXML_Element* deviceElement,
  UPnPRootDevice* rootDevice,
  int maxDepth)

Extracts device information.

Documentation

Extracts device information. Gets the required information from a XML document searching maxDepth deep for any embedded devices and related services under this device

Parameters:
device - pointer to UPnPDevice buffer to store extracted information in
deviceElement - location of device information inside xml dom tree
rootDevice - pointer to the device's root device
maxDepth - indicates the depth level this funtion will search the dom tree for devices embedded within this device
Returns:
None
