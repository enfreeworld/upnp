int UPnP_AcceptSubscription

( UPnPSubscriptionRequest* subReq,
  const GENA_CHAR* subscriptionId,
  UPNP_INT32 timeoutSec,
  IXML_Document* propertySet,
  UPNP_INT32 firstNotifyDelayMsec )

Accept a new subscription request.

Documentation

This function adds a new subscriber device's internal subscriber's list, generates a unique subscription Id for this subscriber, sets a duration in seconds for this subscription to be valid and sends a subscription response indicating success or failure to subscription request.

Parameters:
subReq - address of structure containing subscription request information
subscriptionId - subscription identifier for the subscriber
timeoutSec - duration in seconds for which the subscription is valid
propertySet - pointer to response message in XML format.
firstNotifyDelayMsec - delay in milliseconds before sending the first event notification to the new subscriber
Returns:
error code
